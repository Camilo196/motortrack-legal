# Politica de Privacidad - MotorTrack

Ultima actualizacion: 2026-04-17

MotorTrack es una aplicacion movil para registrar vehiculos, controlar gastos, hacer seguimiento a mantenimientos y administrar recordatorios.

## 1. Informacion que recopilamos

MotorTrack puede recopilar y almacenar:

- correo electronico y datos basicos de autenticacion
- informacion de vehiculos registrada por el usuario
- gastos, mantenimientos, kilometraje y recordatorios asociados a cada vehiculo
- solicitudes de eliminacion de cuenta realizadas desde la app o por soporte

## 2. Como usamos la informacion

Usamos la informacion para:

- autenticar al usuario
- sincronizar sus datos entre sesiones y dispositivos
- calcular estadisticas, costos y recordatorios
- responder solicitudes de soporte, privacidad o eliminacion de cuenta

## 3. Donde se almacenan los datos

Los datos se almacenan en Supabase, servicio utilizado como backend de autenticacion y base de datos.

## 4. Compartimos datos con terceros

MotorTrack no vende informacion personal del usuario.

La app puede apoyarse en proveedores tecnicos para funcionar correctamente, por ejemplo:

- Supabase para autenticacion y base de datos

## 5. Eliminacion de cuenta y datos

El usuario puede solicitar la eliminacion de su cuenta desde la seccion `Cuenta y Privacidad` dentro de la app.

## 6. Seguridad

MotorTrack aplica controles de acceso por usuario en la base de datos para evitar que un usuario acceda a la informacion de otro.

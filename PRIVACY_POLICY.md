# Politica de Privacidad - MotorTrack

Ultima actualizacion: 2026-03-21

MotorTrack es una aplicacion para registrar vehiculos, gastos, mantenimientos y recordatorios.

## 1. Informacion que recopilamos

MotorTrack puede almacenar:

- correo electronico y datos basicos de cuenta para autenticacion
- informacion de vehiculos registrada por el usuario
- gastos, mantenimientos y recordatorios asociados a cada vehiculo
- solicitudes de eliminacion de cuenta realizadas desde la app

## 2. Como usamos la informacion

Usamos la informacion para:

- autenticar al usuario
- sincronizar sus datos entre sesiones
- calcular estadisticas y recordatorios
- atender solicitudes de soporte o eliminacion de cuenta

## 3. Donde se almacenan los datos

Los datos se almacenan en Supabase, servicio usado como backend de autenticacion y base de datos.

## 4. Compartimos datos con terceros

MotorTrack no vende informacion personal del usuario.

La app puede usar proveedores tecnicos para operar correctamente, como:

- Supabase para autenticacion y base de datos

## 5. Eliminacion de cuenta y datos

El usuario puede iniciar una solicitud de eliminacion de cuenta desde la seccion `Cuenta y Privacidad` dentro de la app.

Tambien puede solicitar eliminacion escribiendo a:

`<SUPPORT_EMAIL>`

## 6. Seguridad

Aplicamos controles de acceso por usuario en la base de datos para evitar que un usuario vea datos de otro.
